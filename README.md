# Clean Hamburger Menu Template for LP

LP（ランディングページ）向けの、シンプルで実務を意識したハンバーガーメニューのテンプレートです。  
Vanilla JavaScriptとSCSSを用いて、再利用しやすい構成を意識して実装しています。

---

## 特徴 / Features

- ハンバーガーメニューの開閉
- アイコンのアニメーション（×に変形）
- オーバーレイクリックでメニューを閉じる
- ESCキーでメニューを閉じる
- `aria-expanded` を用いたアクセシビリティ対応
- 関数の抽象化による保守性の高い設計
